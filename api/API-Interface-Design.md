# API 接口设计 V1.0

## 1. 查询五运六气

GET

/api/yunqi/calculate

参数：

dateTime

返回：

YunQiResult

------------------------------------------------------------------------

## 2. 查询年度结果

GET

/api/yunqi/year/{year}

------------------------------------------------------------------------

## 3. 新建问诊记录

POST

/api/intake

------------------------------------------------------------------------

## 4. 获取问诊记录

GET

/api/intake/{id}

------------------------------------------------------------------------

## 5. 生成线索

POST

/api/intake/{id}/clues

返回：

-   六淫线索
-   八纲单项线索
-   时气对应提示

注意：

不得返回诊断。

------------------------------------------------------------------------
