# AGENTS.md

## 项目规则

你正在开发中医五运六气与问诊结构化原型系统。

核心原则：

1.  规则优先 所有五运六气计算必须来自规则文件，不允许自行推导。

2.  领域隔离 yunqi-domain不得依赖React、数据库、API框架。

3.  医疗安全 禁止生成：

-   自动诊断
-   自动辨证
-   方剂
-   中药
-   剂量

允许： - 理论线索 - 规则解释 - 教学提示

4.  时间规则 必须：

-   使用北京时间；
-   使用实际节气交接时刻；
-   不使用固定日期替代节气。

5.  修改规则必须：

-   更新版本号；
-   更新测试；
-   记录变更原因。
